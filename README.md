DistributeSpace-Sketch
=================
Plugin for Sketch that allows to distribute vertical and horizontal space between selected objects.

#Shortcuts
⌘ + ⌥ + ⇧ + V — distribute vertical with space

⌘ + ⌥ + ⇧ + H — distribute horizontal with space
